# 更新履歴 - dialpad-migration-manager

## [2026-03-23] リアルタイム更新の実装とヘッダーレイアウトの修正
- **案件情報のリアルタイム反映**: Supabase Realtime (`postgres_changes`) を導入。案件のステータス変更、追加、削除が全クライアントの画面に即座に反映されるようになりました。
- **ステータス自動遷移ロジックの改善**: 未対応/対応済から未来日付設定時に「対応予定」へ自動変更するロジックを実装。
- **DB更新処理の堅牢化**: `unit_id` ではなく UUID (`id`) を主軸とした更新処理へ変更し、データの整合性を向上。
- **Supabase 設定変更**: `projects` テーブルを `supabase_realtime` パブリケーションに追加し、`REPLICA IDENTITY FULL` を設定。
- **アクティブユーザー表示位置の修正**: `header-content-inner` の幅設定を修正し、ユーザーアイコンを右端に正しく配置。

## [2026-03-13] UI の洗練と動作安定性の向上
- **Ultra Rich ヘッダーの最終微調整**: テーブルヘッダーのテキストサイズを 10px に調整し、スクロール時の固定位置をフィルターパネル下端（256px）に正確に配置。
- **sticky 配置の修正**: `top` オフセットの修正により、重なりや未固定状態を完全に解消。
- **Clear Selection の再配置**: 案件選択解除ボタンを一覧下部へ移動。フィルターバーのレイアウト崩れを解消。
- **一括解除の機能修正**: チェックボックスがクリアされない不具合を修正。

## [2026-03-12] リアルタイムプレゼンスとヘッダー固定化
- **リアルタイムプレゼンス**: Supabase Presence を導入し、右上にアクティブユーザー一覧を表示。
- **ヘッダー完全固定**: プロジェクト一覧のヘッダーがスクロール時に固定されるよう調整。

## [2026-03-12] ライセンス管理のクラウド同期とパフォーマンス高速化
- **ライセンス管理**: ライセンス数の増減・直接編集に対応し、Supabase と同期。
- **共通デザインシステム**: 全ページで共通のヘッダー高さを適用。

## [2026-03-06] Premium UI 全面刷新 & Tailwind CSS v4 移行
- **Tailwind CSS v4 導入**: 最新の Tailwind v4 を採用し、プレミアムなデザインシステムを構築。
- **ダッシュボード刷新**: 統計カードの意匠変更とローディング画面の強化。
- **地理情報連携**: Nominatim を使用した住所から座標への変換（ジオコーディング）機能を実装。
