# 更新履歴 - dialpad-migration-manager

## [2026-03-06] Premium UI 全面刷新 & Tailwind CSS v4 移行 (v18)

### 主要変更
- **Tailwind CSS v4 導入**: `tailwindcss` + `@tailwindcss/vite` をインストール。`vite.config.js` にViteプラグインを設定し、`@import "tailwindcss"` + `@theme` ブロックでカスタムカラートークン（primary/secondary/accent/bg-dark等）を定義。これによりレイアウト崩れを完全解消。
- **Google Fonts をHTMLに移行**: CSS `@import url()` はTailwind v4の`@import`と競合するため、`index.html`の`<link>`タグに移行。
- **不足CSSクラスを全補完**: `glass-card`・`gradient-text`・`text-muted`・`loading-container`・`loading-spinner`・`logo-icon`・`spinner`・`checkbox-v5`・`motion-option-v13`・`premium-btn-primary`・`icon-btn`・カレンダー系クラス群を追加。
- **ダッシュボード統計カードの刷新**: グラデーションボーダー付き `stat-card` に切り替え、各ステータスに対応したカラーアイコンボックスとグロー効果を追加。
- **ローディング画面の強化**: グラスモーフィズム背景・アニメーションロゴ・スピナー・テキストを追加。

### コード品質
- `eslint.config.js`: `scripts/**` を `globalIgnores` に追加（Node.js スクリプトのlint除外）
- `ProjectList.jsx`: `SortIcon` をコンポーネント外に移動（hooks違反解消）、未使用import削除
- `CalendarView.jsx`: 未使用のdate-fns imports を全削除
- `Settings.jsx`: `fetchProfiles` を `useCallback` 化し宣言順序を修正、未使用state変数を接頭辞 `_` 付きに変更
- `AppContext.jsx`: 未使用props `initialData` 削除、`useApp` に eslint-disable コメント追加
- **lint 0エラー・build 0警告** を達成
