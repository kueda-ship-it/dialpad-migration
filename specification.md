# 仕様書 - dialpad-migration-manager

## 機能概要
- Dialpadの移行進捗管理
- Excelデータからのインポート・連動
- カレンダー表示によるスケジュール管理
- 統計ダッシュボード

## 主要コンポーネント
- `Login`: Azure AD (Microsoft) 認証
- `Sidebar`: ナビゲーション管理 (v17 垂直アイコンバー/幅110px)
- `Dashboard`: 移行進捗の統計表示
- `ProjectList`: 移行プロジェクトの一覧管理 (v18 プレミアムデザイン)
- `CalendarView`: スケジュール可視化
- `MapView`: 拠点情報の地図表示
- `RouteFinder`: ルート検索機能
- `Settings`: ユーザー設定およびシステム設定

## UI/UX デザインシステム (v18)
- **デザインコンセプト**: 「奥行きと輝き」をテーマにした超プレミアムUI。多層的なガラス質感と光彩を採用。
- **サイドバー**: 視認性を重視した v17 垂直アイコンバー形式。幅 110px に拡大し、大型アイコンと明瞭なラベルを配置。
- **ボードレイアウト**: メインコンテンツを最大 1400px に制限し、大画面モニターでも中央に寄るよう最適化。
- **タイムスタンプ (v17)**: 単純な日付入力を廃し、グラス質感の専用バッジコンテナ内に配置。スケジュールとしての視認性を大幅に強化。
- **入力UI**: ブラウザ標準の日付入力をスタイリッシュにカスタマイズ。
- **インタラクション**: ボーダーを排し、グラデーションとシャドウ、スケールアニメーションで操作を豊かに表現。

## 案件管理 (ProjectList) 独自仕様
- **検索バー**: v7 仕様の高品位検索 UI。スケールアニメーション付き。
- **ステータス管理**: テーブル一覧上でプルダウン更新。右側に独自グロウ付きステータスドットを表示。
- **対応日編集**: カスタムデザインの `input[type="date"]` により、プレミアムな操作感で直接編集可能。
- **回線表示**: `line_type`と`locker_type`を組み合わせた詳細表示。
- **ボタン**: 極限まで意匠を削ぎ落としたボーダーレスなプレミアムボタン。
- **ライセンス管理**: ライセンス数の `+` / `-` ボタンは、500msの長押しで連続増減（100ms間隔）が可能。
- **高速レスポンス**: すべての更新操作は「楽観的UI更新（Optimistic UI）」によって、データベースの結果を待たずに即座に画面へ反映される。
- **レンダリング効率**: `React.memo` を使用した行コンポーネントと `tbodyAnimKey` による描画最適化。
- **リアルタイム・プレゼンス**: Supabase Realtime Presence を使用し、現在システムを利用中のユーザーを右上に表示。
- **共通固定ヘッダー**: `App.jsx` で共通化されたヘッダー構造により、全ページで高さとタイトル位置を完全に統一。
- **ヘッダー完全固定化**: プロジェクト一覧のタイトルおよび操作パネル（検索・フィルター）を完全固定。
- **Ultra Rich ヘッダー**: 案件名のヘッダー行に超高品位なタイポグラフィ（11.5px / ウェイト950 / 字間拡大）と、ホバー時の浮遊・発光エフェクトを実装。スクロール時もフィルターパネル直下に完全に固定（sticky）され、常に操作性を維持。
- **ページネーション・フル表示**: 50件ずつの分割表示に加え、全件を一括で確認できる「すべて表示」機能を搭載。
- **一括選択解除 (Clear Selection)**: 案件選択を全解除するボタンは、レイアウトの安定性を保つためテーブル下部（ページネーション上）に浮遊感のあるグラスバッジとして配置。
### ProjectList.jsx (v18)
- **Ultra Rich Header**: `.th-label-rich` クラスを導入。`font-size: 11.5px`, `letter-spacing: 0.28em`, `font-weight: 950`, `text-shadow` 加圧により、極限の視認性と美しさを両立。ホバー時に `translateY` と光彩拡大のアニメーションを付与。
- **Sticky Implementation**: テーブルヘッダー (`<thead>`) に `.sticky-table-header` を適用。フィルターパネル (`top: 124px`) の直下に固定されるよう `top: 178px` に設定。
- **Layout Integrity**: フィルターバーにボタンを詰め込みすぎると改行が発生するため、`Clear Selection` ボタンはテーブル下部 (`main` タグの直後) に配置。`AnimatePresence` で選択時のみ浮遊表示。
