# 仕様書 - dialpad-migration-manager

## 機能概要
- Dialpadの移行進捗管理
- Excelデータからのインポート・連動
- カレンダー表示によるスケジュール管理
- 統計ダッシュボード

## 主要コンポーネント
- `Login`: Azure AD (Microsoft) 認証
- `Sidebar`: ナビゲーション管理 (v7 垂直アイコンバー)
- `Dashboard`: 移行進捗の統計表示
- `ProjectList`: 移行プロジェクトの一覧管理 (v7 プレミアムデザイン)
- `CalendarView`: スケジュール可視化
- `MapView`: 拠点情報の地図表示
- `RouteFinder`: ルート検索機能
- `Settings`: ユーザー設定およびシステム設定

## UI/UX デザインシステム (v7)
- **デザインコンセプト**: 「奥行きと輝き」をテーマにした超プレミアムUI。多層的なガラス質感と光彩を採用。
- **サイドバー**: 視認性を重視した v17 垂直アイコンバー形式。幅 110px に拡大し、大型アイコンと明瞭なラベルを配置。
- **ボードレイアウト**: メインコンテンツを最大 1400px に制限し、大画面モニターでも中央に寄るよう最適化。
- **タイムスタンプ (v17)**: 単純な日付入力を廃し、グラス質感の専用バッジコンテナ内に配置。スケジュールとしての視認性を大幅に強化。
- **入力UI**: ブラウザ標準の日付入力をスタイリッシュにカスタマイズ。
- **インタラクション**: ボーダーを排し、グラデーションとシャドウ、スケールアニメーションで操作を豊かに表現。

## 案件管理 (ProjectList) 独自仕様
- **検索バー**: v7 仕様の高品位検索 UI。スケールアニメーション付き。
- **ステータス管理**: テーブル一覧上でプルダウン更新。右側に独自グロウ付きステータスドットを表示。
- **対応日編集**: カスタムデザインの `input[type="date"]` により、プレミアムな操作感で直接編集可能。
- **回線表示**: `line_type`と`locker_type`を組み合わせた詳細表示。
- **ボタン**: 極限まで意匠を削ぎ落としたボーダーレスなプレミアムボタン。
- **ライセンス管理**: ライセンス数の `+` / `-` ボタンは、500msの長押しで連続増減（100ms間隔）が可能。データは Supabase でクラウド管理され、操作停止から500ms後に同期される（デバウンス処理）。
- **高速レスポンス**: すべての更新操作は「楽観的UI更新（Optimistic UI）」によって、データベースの結果を待たずに即座に画面へ反映される。
- **レンダリング効率**: React.memo を使用した行コンポーネントの再利用、および `tbody` の `key` 最適化により、大量のデータでもスムーズなスクロールと更新を実現。
- **リアルタイム・プレゼンス**: Supabase Realtime Presence を使用し、現在システムを利用中のユーザーを右上に表示（自分自身は除外）。
- **共通固定ヘッダー**: `App.jsx` で共通化されたヘッダー構造により、全ページ（Dashboard, Projects等）で高さとタイトル位置を完全に統一。スクロール時も常に固定。
- **ヘッダー完全固定化**: プロジェクト一覧のタイトルおよび操作パネル（検索・フィルター）を `sticky` レイアウトの最適化により完全固定。
- **カレンダー視認性**: 日本の祝日・振替休日判定ロジックを内蔵。土曜日 (Blue)、日曜日・祝日 (Red) を自動で色分け表示し、重要日程の把握を容易に。また、ヘッダーに月ごとの集計（合計、ステータス別）をリアルタイム表示する「スタッツチップ」を搭載。
- **サイドバー・インテリジェント同期**: `localStorage` キャッシュと Supabase プロフィールテーブルをハイブリッドで参照。アバター画像・管理者権限などを、システムの全画面で遅延なく即時反映。
